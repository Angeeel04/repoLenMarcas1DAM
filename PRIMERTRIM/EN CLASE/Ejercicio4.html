MODULOS 1º DAM

LENGUAJE DE MARCAS
Se exploran lenguajes de marcado como HTML, XML y CSS, utilizados en el desarrollo web y la creación de interfaces de usuario. Los estudiantes aprenden a estructurar y diseñar contenido web.
PROGRAMACIÓN
Se centra en la enseñanza de la programación de software, generalmente utilizando lenguajes como Java o C#. Los estudiantes aprenden a crear y diseñar aplicaciones informáticas.
SISTEMAS MICROINFORMÁTICOS
Se centra en los fundamentos de la informática, abordando temas como la arquitectura de computadoras, sistemas operativos y redes. Los estudiantes aprenden sobre el hardware y software subyacentes que respaldan las aplicaciones informáticas.
BASE DE DATOS
Se estudia la gestión de bases de datos, incluyendo diseño, consultas y mantenimiento. Pueden trabajar con sistemas de gestión de bases de datos como MySQL o SQL Server.
ENTORNOS DE DESARROLLO
Se enfoca en el uso de herramientas de desarrollo de software, como entornos de desarrollo integrados (IDE) y control de versiones. Los estudiantes aprenden a trabajar eficientemente en proyectos de desarrollo de software.
